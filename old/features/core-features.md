### Geração Estática
- Sites HTML estáticos a partir de templates
- `hugo server` para desenvolvimento local com LiveReload

### Velocidade
- Builds em milissegundos (ex: 10k páginas em <2s)
```bash
hugo --gc --minify # Build de produção otimizado
```
